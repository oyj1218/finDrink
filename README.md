# finDrink
### 프론트엔드 & 백엔드 개발 약 3개월 정도 배우고 1 번째로 한 팀프로젝트입니다.
3명의 사람들과 했으며, 저는 3명 중 팀장을 맡았습니다
<br><br>
로고 디자인, 이미지 등 전반적인 디자인부터 시작해 프론트엔드, 카카오API 연동, 백엔드도 했습니다.
<br><br>
주로 프론트엔드 위주로 작업했으며 백엔드는 커뮤니티 게시판을 작업했습니다.
